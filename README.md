# icenote
md记事本
# 特点
+ 支持无限分类
+ 支持搜索
+ 支持预览编辑
+ 支持标签
+ 支持多平台
+ 完全开源
+ 支持导出pdf
# 使用技术
+ electron
+ react
+ redux
+ webpack
+ nodejs
+ antd
+ nedb
+ markdown-pdf
+ underscore
